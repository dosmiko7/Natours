## Run Locally

Make sure you have Node and npm installed (https://nodejs.org/en)

Clone the project

```bash
  git clone https://github.com/dosmiko7/Natours
```

Go to the project directory

```bash
  cd Natours
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
